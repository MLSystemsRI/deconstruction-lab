# Phase 3: Floor System Removal

> Deconstruction Lab | ML Systems | Lucent Lens
> Precondition: Roof off (Phase 1), walls set aside (Phase 2), crane still moving
> Skip this phase on single-story homes — go straight to foundation

---

## Sequence Position

```
Phase 1: Roof off (crane lifts sections)           ✓ Done
Phase 2: Walls laid on side (crane sets aside)      ✓ Done — ground crew disassembles later
Phase 3: Floor system out (THIS PHASE)              ← Crane is here
         └─ Brace lower walls BEFORE pulling floor
Phase 4: Repeat Phase 2–3 for each story below
Foundation: Last
```

**Single-story home:** No floor system between roof and foundation. Phase 3 is skipped — after walls are set aside, you're at the slab/crawlspace/basement level.

**Multi-story:** Each upper floor is a diaphragm (like the roof was). Removing it destabilizes the walls below. Brace first, then pull.

---

## The Problem: Diaphragm Loss (Again)

Same issue as the roof. The floor system is a horizontal diaphragm that braces the walls below against racking. Remove the floor → walls below can bow or collapse outward.

```
BEFORE floor removal:           AFTER floor removal (unbraced):

  ┌──────────────┐
  │  FLOOR DECK  │  ← braces       Walls can rack outward
  ├──────────────┤     walls        ↙                    ↘
  │              │    below       /                        \
  │    WALLS     │               │        WALLS            │
  │              │               │                         │
  └──────────────┘               └─────────────────────────┘
```

**Rule: Brace the lower-story walls BEFORE removing the floor above them.**

---

## Bracing the Lower Walls

### If lower walls have NOT been removed yet (typical multi-story sequence)

| Brace Type | Where | Material | When to Remove |
|---|---|---|---|
| Diagonal kickers (exterior) | Every 8' along exterior walls, plate to ground stake or deadman | 2×6 | After lower walls are set aside |
| Horizontal ties (interior) | Corner to corner across the floor below | 2×6 on edge, temporary screwed | Last — during lower wall removal |
| Lateral bracing at openings | Large window/door openings on lower floor | 2×4 X-brace across opening | With lower wall removal |

**If the lower walls HAVE already been done** (e.g., ground floor walls already set aside on a 3-story): You're at the foundation. No bracing needed — floor system is the last structural element above foundation.

## Crane Lift Strategy — Floor Sections

Same logic as the roof: cut into sections, rig, lift, set down, ground crew disassembles later.

### Section Sizing

| Approach | Section Size | Weight | Lifts Needed |
|---|---|---|---|
| **Full-bay sections** (cut at bearing walls) | 12–16' × 20–25' | 2,500–4,500 lbs | 4–6 lifts |
| Half-bay sections | 8–12' × 12–16' | 1,200–2,500 lbs | 8–10 lifts |

**Recommended: Cut at bearing wall lines.** The floor naturally spans between bearing walls — cut lines follow the structure. Each section stays rigid during lift because the joists are still connected to the subfloor.

### Cut Sequence

1. **Remove finish flooring first** (if valuable — hardwood yes, carpet no)
   - Hardwood: ground crew pulls while crane does walls. Or leave attached and separate on ground.
   - Carpet/vinyl: strip and trash. Low/no recovery value.
   - Tile: leave attached to subfloor — too labor-intensive to save in place.

2. **Cut subfloor along joist lines** (circular saw, set depth to subfloor thickness only)
   - Cut AT a joist so each section has edge joists for rigidity
   - Same principle as cutting roof at rafter lines

3. **Cut joists at bearing points** (recip saw)
   - Joist-to-beam or joist-to-bearing-wall connections
   - If joist hangers: cut nails or unbolt. Hangers are reusable if unbolted.

4. **Rig section** (slings under joists, spreader bar above)
   - Floor sections are heavier than roof sections but more rigid
   - Sling at 1/3 points along the span

5. **Lift and set in staging area** — ground crew disassembles later

### Rigging

```
         CRANE HOOK
             │
       ┌─────┴─────┐
       │ SPREADER   │
       │    BAR     │
       └──┬─────┬──┘
          │     │
     SLING│     │SLING
          │     │
   ═══════╧═════╧═══════
   │    SUBFLOOR (top)    │
   │ ──────────────────── │  ← joists hanging below
   │ │joist│joist│joist│  │
   ═══════════════════════
```

Floor sections are lifted subfloor-up (joists hanging). Set down the same way — subfloor on ground, joists up. Ground crew works from the top down: subfloor sheets off, then joists.

## Hardwood Flooring — Pull BEFORE Crane Touches Floor

Hardwood is the highest per-SF value material in the house. Ground crew pulls it while crane is still setting wall panels aside.

| Type | Recovery Rate | Resale | Per Home (800 SF hardwood) |
|---|---|---|---|
| Oak strip (3/4" solid) | 70–90% (if careful) | $3–$6/SF | **$1,700–$4,300** |
| Pine wide plank | 80–95% | $4–$8/SF (antique premium) | **$2,600–$6,100** |
| Engineered hardwood | Low — delaminates | $0–$1/SF | Minimal |

**Timing:** Crew finishes wall panel disassembly in staging area → moves inside to pull hardwood → crane finishes walls → crane starts floor cuts. Hardwood is out before the floor moves.

## Ground Disassembly — Floor Sections

Floor sections are set on their sides in the staging area (same as wall panels). Ground crew should be done with walls by the time floor sections start arriving.

**Sections on their sides** — gives crew access to both subfloor face and joist face without crawling under. Lean against sawhorses, blocks, or wall panels already staged.

| Order | Component | Method | Recovery |
|---|---|---|---|
| 1 | Subfloor sheets | **Preserve the subfloor.** Cut nails with oscillating tool (flush-cut between joist and sheet). Same technique as roof sheathing — extra 2–3 min per sheet saves the material. | 50–70% intact |
| 2 | Bridging/blocking | Pull or cut from between joists. | 80%+ |
| 3 | Joists | Now free-standing. Stack by size/length. | **85–95%** |
| 4 | Rim joist | Pull from ends. | 70–85% |
| 5 | Joist hangers/hardware | Unbolt or pry. | 50–70% reusable |

### Floor Joists = High-Value Recovery

Floor joists are typically the best lumber in the house:
- Long, straight, dimensional (2×10, 2×12)
- Protected from weather (interior)
- Less nail damage than wall studs (fewer penetrations)
- **Resale: $8–$20 per joist** depending on species/size/length
- 50 joists × $12 avg = **$600** in lumber alone

## Timeline

| Task | Who | Duration | Notes |
|---|---|---|---|
| Crew finishes wall panel disassembly | Ground crew | Ongoing from Phase 2 | Should be wrapping up |
| Pull hardwood flooring | Ground crew (inside) | 1–2 hrs | While crane finishes walls |
| Brace lower walls | Ground crew | 30–45 min | Before any floor cuts |
| Cut floor into sections | Roof cutter | 30–60 min | Circular saw (subfloor) + recip saw (joists) |
| Rig + lift + set sections on sides | Crane + rigger | 60–90 min | 4–6 lifts, ~15–20 min each |
| Ground crew disassembles floor sections | Ground crew | 2–3 hrs | Subfloor preserved, joists stacked |
| **Crane time for floor** | | **~2.5–4.5 hrs** | Crane moves to next story or foundation |

**Parallel flow:** Ground crew pulls hardwood → crane cuts floor → sections go to staging on their sides → crew (done with walls) starts floor disassembly → crane is already moving down.

## Safety

| Hazard | Control |
|---|---|
| Floor collapse during cutting | Cut from stable platform. Never stand on an isolated section. |
| Falling through openings | After sections removed, lower floor is exposed. Barricade/cover all openings. |
| Lower wall instability | Bracing installed BEFORE floor removal. Verified by lead. |
| Heavy sections (up to 4,500 lbs) | Crane lift plan updated for floor weights. No manual carries. |
| Nail hazard (subfloor/joists) | Magnetic sweep staging area after each section set. |

## Connection to Overall Sequence

```
Crane (never stops):
  Roof sections → Wall panels on sides → Floor sections on sides → [repeat] → Foundation

Ground crew (one phase behind crane):
  Phase 1: Roof disassembly     │ Phase 2: Pull hardwood inside
  Phase 2: Wall disassembly     │ Phase 3: Floor disassembly
  Phase 3: Floor disassembly    │ Phase 4: ...

Staging area (buffer between crane and crew):
  Wall panels on sides ──→ drywall stacked, lumber stacked, sheathing stacked
  Floor sections on sides ──→ subfloor preserved, joists stacked
  Roof sections ──→ shingles stockpiled, rafters stacked
```

**Everything goes on its side in the staging area. Crane keeps moving. Crew catches up.**
